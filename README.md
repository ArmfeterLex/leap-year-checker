# Leap Year Checker

Программа на Python для проверки года на високосность.

## Функциональность

- Ввод года
- Проверка по правилам григорианского календаря:
  - Год високосный, если кратен 4 и не кратен 100
  - Год високосный, если кратен 400
  - В остальных случаях — невисокосный

## Пример работы

Введите год: 2020
Год високосный

Введите год: 2100
Год невисокосный

Введите год: 2000
Год високосный

## Куда можно встроить

- В календарные приложения
- В расчёт количества дней в году
- В учебные материалы по условным операторам
- В задачи по математике и программированию

## Требования

- Python 3.x

## Примечание

Программа использует стандартные правила григорианского календаря. Для юлианского календаря логика будет другой.
